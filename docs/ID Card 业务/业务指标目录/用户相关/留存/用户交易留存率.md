### 业务定义

* 将用户按照首次交易所在月进行分组，然后分别查看每组用户在后续自然月(Mob N)的是否有交易产生，如果产生，则记该用户为留存用户数。 
* 某个Mob N的用户留存率 = 在该Mob N 的留存用户数 / Mob 0 中的全部用户数。
* 交易指Auth Approved。
